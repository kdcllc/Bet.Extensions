Add the following to the project

```csharp
    dotnet add package Bet.Extensions
```

## Usage

The following classes to support `Async` development:

- `ActionOrAsyncFunc`
- `AsyncExpiringLazy`
- `AsyncFunc`
- `AsyncLock`
- `ReloadToken`
